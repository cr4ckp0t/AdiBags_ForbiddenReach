# AdiBags - ForbiddenReach
A simple AdiBags filter for items found in The Forbidden Reach (Patch 10.0.7)

https://www.curseforge.com/wow/addons/adibags-forbiddenreach

## Missing an item or Found a Bug?

Simply [open a ticket](https://github.com/cr4ckp0t/AdiBags_ForbiddenReach/issues) and I will get it added or fixed as soon as possible.
